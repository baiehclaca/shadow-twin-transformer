The Core Concept

Shadow Twin extends the standard transformer with parallel processing streams that work together with external memory, enabling more robust reasoning and interpretable predictions.

Dual Streams

Text and shadow streams evolve in tandem, each processing information while exchanging insights. The shadow stream provides internal reflection and self-checking.

External Memory

Persistent memory buffer stores representations and context throughout generation, enabling the model to retrieve and reason over historical states with temporal awareness.

Confidence-Aware

Output generation produces both predictions and confidence scores. When streams diverge, the model expresses uncertainty and provides alternative hypotheses.
